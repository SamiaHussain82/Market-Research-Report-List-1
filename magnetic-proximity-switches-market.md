<p><ul><li>Reed Type：With Actuating Magnet</li><li>Hall Effect Type : Solid State Output</li></ul></p>
<p><p>Magnetic proximity switches are used for detecting the presence of an object without direct contact. The Reed type switch operates with a magnet; when the magnet approaches, it closes the internal contacts, allowing current to flow. Conversely, the Hall Effect type uses a solid-state output to sense magnetic fields, producing a signal when a magnet is nearby. Both types are commonly employed in automation, security systems, and industrial applications, offering reliable and efficient non-contact sensing solutions.</p></p>
